BASIC HTML LIST

Html list helps developers to group a set of related items in lists.

HTML UNORDERED LIST

The unodered list usually starts with ul tag. each list contains the li tag, marked with small black circles called; (bullets).

  • Coffee
  • Tea
  • Milk

HTML ORDERED LIST

The unodered list usually starts with ol tag. each list contains the li tag, marked with numbers by default.

  1. Rice
  2. Beans
  3. Yam

HTML DESCRIPTION LIST

Html description list is a list of items with the description of each term. usually starts with dl tag which defines the description list, dt tag defines the term (name) while the dd tag defines each term.

Coffee
Black hot drink
Milk
White cold drink
