HTML Form Element

The HTML form element can contain one or more of the following forms elements:

INPUT ELEMENT

LABEL/SELECT/OPTION ELEMENT

The {label} element is useful for screen readers users because the screen-reader will read out loud the label when the user focus on the element, {Select} element defines a drop-down list While {Option} element defines an option that can be selected.


TEXTAREA ELEMENT

Textarea elements defines a multi-line input field

BUTTON

Button elements defines a clickable button.

FIELDSET and LEGEND ELEMENT

Fieldset element is used to group related data in a form While Legend element defines a caption for the fieldset element

Personalia:




DATALIST ELEMENT

Datalist element specifies a list of pre-define options for an input element and it enanble users to see a drop-down list of the pre-defined options as they input data. The list attribute of the input element must refer to the id attribute of the datalist.

OUTPUT ELEMENT

Output element represents the result of a calculation.
